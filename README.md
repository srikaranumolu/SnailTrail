**Description**
Snail Trail is a 2D Pixel Art platformer game. You play as Pink Man, who needs to avoid the snails and reach the checkpoints.
This game was created by Srikar A using Godot 4. 
All assets used in this game were created by pixel frog and are from this itch.io page: https://pixelfrog-assets.itch.io/pixel-adventure-1
This live demo for this game can be found at: https://srikara8.itch.io/snail-trail

**Game Breakdown**
The game has 4 main sections: the main menu, level 1, level 2, and level 3
In each of these sections, including the main menu, the player can move and jump around freely.
The goal of the game is to avoid all the snails and reach the flags while collecting as many apples as possible.
Midway through the game, Pink Man unlocks a hidden talent that assists him in completing the game.
The main menu has 5 buttons. The play button is used to start the game and begin on Level 1. The settings button is used to adjust the game's volume. The restart button is to restart the main menu scene and animations. The leaderboard button can be used to see your best scores for each level. The levels button can be used to select a specific level if you don't want to start at level 1.

**Controls:**
Use W or Space to jump, A to move left, and D to move right. Press jump (W or Space) twice to double jump when you unlock it.
