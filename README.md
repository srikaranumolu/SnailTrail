Snail Trail is a 2D Pixel Art platformer game. You play as Pink Man who needs to avoid the snails and reach the checkpoints.
This game was created by Srikar Anumolu. 
All assets used in this game were created by pixel frog and are from this itch.io page: https://pixelfrog-assets.itch.io/pixel-adventure-1
I used Godot 4 to make this game.
